# dev-hobbes-cli-hello-world

## Description
A POC for hobbes programming language.

## Tech stack
- g++
- make
- cmake
- llvm
- python

## Docker stack
- ubuntu:22.04
- rzuckerm/hobbes

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
